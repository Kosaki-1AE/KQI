- 入力（時系列ログ）  
    t（時刻）, x(t)（観測・スコア・自己評価など）, r(t)（責任ベクトル：重み/方向性）, ctx（タグ：dance/child/invest/other）
- 責任フィルタ：$x(t)$ を $r(t)$ で重み付けし、責任込み信号 $y(t)=r(t)·x(t)$ を作る
- 周波数分解（STFT/FFT）： 
	- 低周波帯 = 長期（子育てとか投資とかの「長期的なもの」がこれ）
	- 中帯 = 中期
	- 高周波帯 = 短期（即興ダンスなどの「短期的なもの」がこれ）
- 指標  
	- ```Stillness = P_low / (P_low + P_high)```（低周波優勢度）
	- ```Motion = P_high / (P_low + P_high)```（高周波優勢度）
	- ```Interference``` = 相互情報っぽい混合度（帯域間の相関/位相差の整合）
- フィードバック学習  
	- 短期：直近の報い（例：観客反応/自己満足/小利確）で r(t) と行動ポリシーを微調整
	- 長期：累積報い（成長/自立/含み益）で低周波寄りの重み・しきい値を更新
- 意思決定  
	- ```Motion```優勢→“攻め”（出す/当てる/密度を上げる）
	- ```Stillness```優勢→“待つ”（抜く/間合い/保有）
	- ```Interference```高→“同期/共鳴アクション”（キメ/間での大技/ポジション調整）
