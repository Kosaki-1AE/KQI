#計算機科学/Programming 
一応外部ストレージ作ったんで共有。実装元はGALLERIAのUbuntuでございます。
通常はこれね↓
![[Ubuntu_selfstrage]]

複数人が登録する場合↓
![[Ubuntu_anystrage]]

Samba設定ファイル編集。まずは個人用。
```bash
[share]
   path = /etc/Share
   browseable = yes
   read only = no
   guest ok = no
   valid users = Society5
   force user = society5
   create mask = 0664
   directory mask = 0775
```
続いて複数人の方ね。
```bash
[share]
   path = /etc/share
   browseable = yes
   read only = no
   guest ok = no
   valid users = @smbshare
   force group = smbshare
   create mask = 0664
   directory mask = 2775
```

続いて、Sambaユーザー登録
```bash
sudo smbpasswd -a Society5   # 登録
sudo smbpasswd -e Society5   # 有効化
```
なので再起動してセット完了
```bash
sudo systemctl restart smbd
```

書き込み権限はこうする↓
```bash
chown -R Society5:Society5 /home/sasaki/Share
chmod -R 775 /etc/Share
```
Windowsでキャッシュ削除したいとき↓
```bash
net use * /delete
```

Wi-Fi利用時は同じSSIDで接続することが必須ね。固定IPを設定しておくと便利やで。

Windows側から入ろうとするとこうなる↓
```bash
\\<UbuntuのIPアドレス(ip aでこれを確認できるっす)>\share
```
