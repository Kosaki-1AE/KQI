# 🌌 ノリエントロピー理論（Nori Entropy Theory）全体まとめ（ver.0.1）

## 🧠 理論の主旨

**ノリエントロピーとは、熱力学的エントロピーに「意味」「責任」「即興性」を導入し、  
カオスの中に“ノリ”や“共鳴”を生む現象を数理的に扱う理論である。**

この理論は、人間の行動・空気感・感情の変動を、  
従来の物理法則とは異なる“意味のある揺れ”として捉える拡張モデルであり、  
HCI・感情研究・即興芸術などに応用可能。

---

## 🔩 基本構成要素と数式

| 記号 | 意味 | 役割 |
|------|------|------|
| `H(t)` | エントロピー | 状態の揺れ・可能性の分布。空気の乱雑さ。 |
| `S(t)` | Stillness | 秩序・静止状態。エントロピーの“落ち着き”。 |
| `Rself` | 自己責任 | 意図と意志。自分から矢を放つ力（意味付け）。 |
| `Renv` | 環境圧 | 空気の重さ。外からの抑圧や制約。 |
| `n` | 習熟度 | 経験・即興力。空気を読んで矢を打つ精度。 |
| `I(t)` | 即興性 | 状況変化への反応能力。柔軟性＋瞬発力。 |
| `A(t)` | 行動波 | 実際のノリ・動き・ツッコミ・表現。 |
| `dH/dt` | 揺れの変動 | ノリのタイミング（刺さる瞬間）を示す。 |
