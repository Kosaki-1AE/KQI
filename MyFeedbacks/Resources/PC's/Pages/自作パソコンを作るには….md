まずは一番の基本となる「ソフトウェア」と「ハードウェア」の関係性についてです。
ソフトウェア：コンピュータの中身
ハードウェア：

まあまあ話題に挙がるのがソフトなんで、ソフトウェアについて色々書いておこうかと。
BIOS：Basic Input Output Systemの略。起動して最初に現れるソフトウェアである。CUIを利用して進めていく
OS：Operating Systemの略で、パソコンの全ソフトウェアをコマンド等を用いて操るシステムのこと。パソコンの製作会社によって様々な種類が存在する
UNIX系：Red Hat Enterprise Linuxなどがある。AT&Tベル研究所で開発されたものがほとんど。サーバーのライセンス料がかからないため、よくサーバーを立てる場合に用いられることが多い。ちなみにMacOSやiOSも含まれるが、こちらはAppleが一体型での発売を行ったため別とする
Windows系：Microsoft社が開発しているもので、サーバーを立てるものや個人向けなど色々含まれる

システムコール：OSが元来持っている機能を発揮するための機構。大体の場合はC言語の形で表現される

コンピュータ：広義的には計算やデータ処理を自動的に行う装置全般のことだが、今はパソコンのことを指すようになった
パソコン(PC)：Personal Computerの略で、個人によって使用されるコンピュータのこと。スマホもこれにあたる

UI：User Interface(ユーザーインターフェース)の略。人間と製品やサービスを繋ぐ接点となるもの。Webサイトやアプリケーションがこれにあたる

CUI：Character User Interface(キャラクターユーザーインターフェース)の略で、Mac以前のPCの画面のこと。CLI(Command Line Interface)やCUI(Console User Interface)とも言う。要はコマンドプロンプトから打ち込んでソフトウェアを動かすという仕組み

GUI：Graphical User Interface(グラフィカルユーザーインターフェース)の略で、Mac以降のPCの画面のことで、アイコンを押すことでソフトウェアを起動できる仕組み

カーソル：GUIになったときにソフトウェアのアイコンをクリックするために開発された記号やマークのこと


CPU：Central Processing Unit(中央処理装置)の略で、コンピュータの脳となる部分。様々な部分で構成されている。ほとんどのノートパソコンでintel製のものを導入している
クロック周波数：CPUが処理を行う時に発している信号を扱う速さを示すもので、単位は『Hz』。この数値が大きくなればなるほど性能が良いと判断できる。1Hzで1秒間に1回振動していることを表す。同時に発生するのは音と熱。これの対策としてファンや冷却器などがついている
プロセッサー：コアと呼ばれるもの。人間でいう『The 脳』と言える場所でコンピュータに命令を行う場所。これの個数であるコア数が多ければ多い程性能が良いと判断できる
スレッド：コアの中で同時に処理できる命令の最大数を表す。2ちゃんねらーは聞きなじみがあるだろうが、2ちゃんにおけるスレと同じ意味と考えて差し支えない
マザーボード(マザボ)：パソコンの神経となる部分で、パソコンのハードウェア側の一番基礎となるもの。これがないとコア(CPU)などをそもそも組み合わせられない

RAM：Random Access Memory（揮発性メモリ）の略で別名はメインメモリ（主記憶装置）。電源を消したらデータが消えるメモリなので、ドライブへの保存は絶対必須になる。だからこそWordとかExcelとかだとCtrl+Sで保存をする

ROM：Read Only Memory（不揮発性メモリ）。電源消えてもデータ消えないメモリ

キャッシュメモリ：よく使うデータを保存、データの読み書きの速度を速くするメモリ

フラッシュメモリ：データの書き換えや消去を電気的に行う不揮発性のメモリ

ドライブ：ファイルやデータを入れるもの

マウス：カーソルを操るための機器。形がネズミに似ていたことで『Mouse』と名付けられた

エクスプローラー：最初から入っているアプリケーションの一つ。フォルダやアプリなどの保存場所でもある

ディレクトリ：フォルダのこと。階層構造の一番上をルートディレクトリ、現在進行中のフォルダをカレントディレクトリ(ワーキングディレクトリ)という。エクスプローラーを開いた時(Excelとかの保存先)のアドレスバーを開いた時に表示されているもの

パス：目標のデータまでの道のり。ルートディレクトリから目標のデータまでは絶対パス、逆を相対パスという。これに関しては『橋』を用いたほうが分かりやすく、絶対パスでは『絶対的な』順番でなければ橋が通じないが、相対パスは向こう岸のみであるため、どのパスでも通じれば必ず開くという仕組みである
アドレス：データの居場所を示すもの。文字列や番号、ビット列などの識別符号で表されている
URL：Uniform Resource Locaterの略で、siteの目印になるもの。パスの正体
拡張子：URLの末尾についている何のファイルかを判断する目印のようなもの。詐欺サイトなどを見破れることもある

マスタファイル：データベースの中での一番の元になっているファイルのこと

プログレスバー：スマホとかでアプリをインストールしてるときに表示されてるあの円が完成するみたいな動きのこと。パソコンでは横棒状になってるはず

半導体メモリ：半導体で作られる記憶媒体。電力で読み取る
磁気ディスク：磁気の力で書き込む記憶媒体
光ディスク：レーザー光線を当てて書き込む記憶媒体

CD：Compact Discの略で、音楽を保存する媒体。記憶容量は約700MB
DVD：Digital Versatile Discの略で、動画を保存する媒体。記憶容量は4.7GB
USB：Universal Serial Busの略。ドライブの一つ容量は最低1GB、最大1TB
SD：Secure Digitalの略で、データを保存する媒体
SSD：Solid State Driveの略。データ転送速度が早い代わりに小容量となっている。小容量とはいえUSBとは段違いに入る。最低128GB。俗にいうキャッシュメモリ
HDD：Hard Disk Driveの略。データ転送速度が遅い代わりに大容量となっている。俗にいうフラッシュメモリ

NIC：Network Interface Cardの略で、LAM接続のための拡張装置のこと。基盤とセットになっており、これをまんま使う

転送速度
USB1.1：初代USB。速度は12bps
USB2.0：2代目USB。速度は480Mbps
USB3.0：3代目USB。速度は5Gbps
USB3.1：4代目USB。速度は10Gbps
USB3.2：5代目USB。速度は20Gbps

コネクターの種類
Type-A：もういうまでもない、『USB』といえばっていうやつ
Type-B：スキャナーやプリンターとかでみるやつ
Type-C：俗にいうiPhone15の充電ケーブル
Mini-USB Type-B：あんまみないやつ
Micro-USB2.0 Type-B：デジカメとかの充電器でみるやつ
Micro-USB3.0 Type-B：これもあんまみないやつ

こんなの付けられるのも夢やね。
Plug and Play：最近のOSに入っている機能で、自動的にデバイスドライバが起動する

マルチタスク：1台で複数のソフトウェアを同時に実行する機能

デジタルハブ：パソコンの中にデジタル関連を全て入れるという仕組み。動画や画像なども全てパソコンの中に入れられるようにした仕組みがこれ
