## 📐 ノリエントロピー理論の公式一覧（ver.0.1）

---

### ① 行動波（ノリ）の発生式
A(t) = I(t) × [n × Rself - Renv × H(t)]

| 記号      | 意味                |
| ------- | ----------------- |
| `A(t)`  | 行動波（ノリの発現）        |
| `I(t)`  | 即興性（瞬時の柔軟反応力）     |
| `n`     | 習熟度・経験値           |
| `Rself` | 自己責任ベクトル（内発的意図）   |
| `Renv`  | 環境圧（空気の重さ、制約）     |
| `H(t)`  | エントロピー（空気の乱れ、情報量） |

- ノリの強さは「即興力 × 責任 × 習熟」で決まり、  
  空気が乱れて重いと（H(t)↑, Renv↑）発動しにくい。

---

### ② Stillnessスコア（止まりの質）
S(t) = (1 - H(t)) + stability(dH/dt)

| 記号 | 意味 |
|------|------|
| `S(t)` | Stillness（止まりの力、空気を待てる静寂） |
| `H(t)` | エントロピー（空気の乱れ） |
| `dH/dt` | エントロピーの変化率（揺れの激しさ） |
| `stability()` | 安定度（変化がなだらかかどうか） |

- 空気が静かで（H(t)が小さい）、変化が穏やか（dH/dtが安定）なら、  
  次の一手が“刺さる”最高の間になる。

---

## ③ エントロピー変動（気づきの兆し）
dH/dt
- 空気（エントロピー）の乱れの時間変化。
- 情報のズレ・感情の揺れ・ノリの切り替えなどを示す重要パラメータ。
- 感情ジャーナルや空間UIでの「気づき」瞬間の抽出に使える。
