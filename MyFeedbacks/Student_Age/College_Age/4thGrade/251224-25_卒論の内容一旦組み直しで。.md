はじめにの部分ね⇩
近年、大規模言語モデルの台頭や2024年パリ五輪におけるブレイキン競技化を契機として、ダンス表現に対する社会的関心が高まっている。これに伴い、ダンスを教育や分析の対象として扱う機会も増加しており、AIによるダンス理解や評価の可能性が注目されている。既存のダンス解析研究では、姿勢推定や動作分類など、再現性の高い振付的ダンスを対象とした手法が多い。一方で、音楽や場の状況に即応して生成される即興ダンスは、正解ラベルを持たず、判断過程が内部的であるため、AIによる解析が困難である。AIがこのような即興的判断を扱うためには、結果としての動作だけでなく、判断に至る過程、すなわち「努力の軌跡」を構造的に扱う必要があると考えられる。本研究では、この軌跡をフィードバックとして記録・整理することが重要な鍵であると捉えた。即興ダンスは歴史的にも、音楽や他者との相互作用を通じて更新される文化として発展してきた。このような相互作用的な判断構造は、サイバネティクスにおけるフィードバック制御の概念と整合的に理解できる。そこで本研究では、即興ダンスの実践経験に基づく内省的フィードバックログを用い、即興ダンスにおけるトランジションの判断過程をサイバネティクス的視点から整理・記述可能な生成アルゴリズムとして提案することを目的とする。


ここからやるのが研究手法のところで定義する奴ら⇩
サイバネティクスでの定義⇩
なお本研究における即興的生成とは、あらかじめ固定された動作列を出力することではなく、環境入力および内部状態に基づいて状態遷移を逐次更新する過程を指す。

トランジションの定義⇩
本研究では、トランジションを即興ダンスにおける重要な構成要素として位置づける。本研究におけるトランジションとは、即興ダンスにおいて、音楽、身体状態、および場の状況といった複数の入力情報に基づき、動作の継続、変更、あるいは停止といった選択を行う判断および調整の過程を指す。この定義において、トランジションは単なる動作間の接続ではなく、ダンサー内部における判断構造および状態遷移として捉えられる。したがって、本研究が対象とする即興的生成とは、固定された動作列を出力することではなく、トランジションを逐次更新し続ける過程そのものを生成することを意味する。

- 意味のあるコスト：伏線回収が約束されてるもの(そもそもの契約時の前提条件に**期限付きの範囲**が含まれてるので、どこまでで調子を狂わせれば良いのかが確定する＝面白さが確定で存在するってことになる)

0が意味のない失敗、0.5が意味のある失敗の方ですわ。

- n = 1（3¹ = 3）
	- 単発判断
	- 今やる / やらない / 保留

	- 学習が始まる最小単位
- n = 3（3³ = 27）

	- **0–27ゾーン** / ここがあなたの言う **0–30 ≒ 0–27** と一致する。
	- 「迷ってるけど、まだ設計がない」段階
- n = 4（3⁴ = 81）
	- 行動 × 結果 × フィードバック × 俯瞰

	- 等身大を保った即興・創作・研究が可能
- n = 5（3⁵ = 243）
	- 設計の設計
	- メタ制約の操作 / ほとんどの人は**無意識でしか触れない**領域。
	- 天才・設計者・文化側の視点

**GSMC = n 制御ロジック**らしいね。

- **3^n**：状態数・演算空間・制御の話
- **GSMC**：軸の意味・選択・構造の話
- 両者は：**上下関係じゃなく直交**
